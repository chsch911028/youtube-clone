## Youtube Clone 프로젝트

### 기술 스택

- React
- Redux
- Redux-Saga
- HTML, CSS, JavaScript

### 프로젝트 실행 방법

- yarn start

### 구현 내용

1. 메인 화면 전체 레이아웃 구성

   - Nav
   - Main
   - SideBar

2. NavBar 구성요소 구현

   - MenuIcon
   - LogoIcon
   - SearchBar
   - SearchIcon
   - VideoIcon
   - CategoryIcon
   - ShareIcon
   - NotificationIcon
   - MeIcon

3. Main 구성요소 구현
   - Banner
   - BannerVideo
   - BannerDescription
   - Contents
   - ContentsTitle
   - VideoList
   - VideoItem
